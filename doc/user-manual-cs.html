Krok 1

Stáhněte si nejnovější verzi Incomaker pluginu.

Krok 2

Jděte na záložku Plugins a klikněte na tlačítko Add New.

Klikněte na Upload Plugin a poté vyberte soubor incomaker.zip, který jste právě stáhnuli.

Krok 3

V sekci Installed Plugins najděte Incomaker klikněte na Settings.

Krok 4

Vyplňte Váš API Key a klikněte na Save Changes. Pokud jste vložili správnou hodnotu, Account ID a Plugin ID se doplní automaticky.

Jak získat API Key

Hodnotu Vašeho API Key naleznete ve svém Incomaker účtu po kliknutí na položku Eshop info.

Otevřete stránku my.incomaker.com

V pravém horním rohu klikněte na obrázek portrétu a zvolte možnost Eshop info

Zkopírujte hodnotu políčka API key

Zkopírujte API Key do WooCommerce a uložte. Pokud jste vložili správnou hodnotu, políčka Account ID a Plugin ID se doplní automaticky.

Připojení WooCommerce je nyní hotové a následuje automatická synchronizace dat.
